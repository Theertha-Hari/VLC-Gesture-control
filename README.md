# VLC Gesture Control

This project allows you to control the **VLC media player** using **hand gestures** detected in real-time via your webcam. It leverages computer vision to map gestures to VLC commands.

---

## Features

- Real-time hand gesture detection using **MediaPipe**  
- Control VLC playback: Play, Pause, Speed 
- Lightweight Python application using **OpenCV**  

---

## Tech Stack

- **Python**  
- **OpenCV**  
- **MediaPipe**  
- **pyautogui** (for simulating keyboard commands to VLC)  

---

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/VLC-Gesture-Control.git
cd VLC-Gesture-Control
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the script:

```bash
python main.py
```

4. Use your hand gestures in front of the webcam to control VLC.
